## 1. 区間数の物理的導出

```
セクション長 = c / (2 × fs)
区間数 N = 声道長 / セクション長 = 2 × L × fs / c
```

| fs (Hz) | セクション長 (cm) | 必要区間数 N |
|---------|------------------|-------------|
| 8,000   | 2.19             | 8           |
| 16,000  | 1.09             | **16**      |
| 22,050  | 0.79             | 22          |
| **44,100** | **0.40**      | **44**      |
| 48,000  | 0.37             | 48          |

N=16 で fs=44100Hz を使うと声道長が約6.3cmになり、子供の声道より短くなる。Pink Trombone も同様に44区間を使用。

## 2. Kelly-Lochbaum アルゴリズムの数学的基盤

### 2.1 コアアルゴリズム (Smith 式 1乗算接合)

各接合部 k (k = 0, ..., N-2) での散乱計算:

```
delta = r_k × (f_k[n] - b_{k+1}[n])
f_{k+1}[n] = f_k[n] - delta
b_k[n]     = b_{k+1}[n] + delta
```

反射係数: `r_k = (A_{k+1} - A_k) / (A_{k+1} + A_k)`

Smith (1985) の1乗算形式により、乗算回数を2→1に削減 (43接合×1乗算/サンプル)。

### 2.2 境界条件

| 境界 | モデル | 根拠 |
|------|--------|------|
| 声門端 | 体積速度源 + 反射 r≈0.95 | 声門インピーダンスは声道より大きい |
| 唇端 | 反射 R_L≈-0.85 | 管端→自由空間の音響インピーダンス不整合 |
| 放射特性 | 1次差分 output = f_N[n] - α×f_N[n-1] | Flanagan (1972) の唇端放射モデル近似 |

### 2.3 壁面損失の物理的根拠

声道壁の粘性・熱伝導による音響エネルギー損失。mu≈0.999 は軟組織特性から経験的に決定。無損失だと高次倍音が過剰に残り金属的な不自然な音になる。

## 3. 声門音源モデルの選択理由

- **KLGLOTT88**: 多項式ベース (`6.75 * t²(1-t)`)、暗黙方程式不要、4-5 ops/sample。AudioWorklet のリアルタイム制約に適合。
- **LF (Liljencrants-Fant 1985)**: `E(t) = E0·exp(αt)·sin(ω_g·t)` + 指数回復相。Rd 一つでチェスト↔ファルセット連続制御。α の求解に Newton-Raphson が必要。
- **却下案**: Rosenberg-C, KLATT88 波形 — パラメータの音声学的解釈性で LF に劣る。

### 位相アキュムレータ方式

`phase += F0/sampleRate` (毎サンプル)。F0補間は対数空間で行う（知覚的に均一: 半音=等比数列）。有声/無声混合: `output = AV × pulse(phase) + AN × noise()`、切替時5〜10msクロスフェード。

### 4.3 チューニング指針

- F2 を下げたい → 口腔後部を狭める（後舌的に）

## 5. フォルマント推定の理論的背景

断面積 → 反射係数 → LPC 多項式 → コンパニオン行列 (43x43) の固有値 (QR 分解, ~1-3ms) で遅延ゼロのフォルマント求解が可能 (step-up procedure)。実装では伝達行列方式を採用したが、LPC ベースの旧コードもフォールバック用にエクスポート維持。

---

## 6. レイテンシ見積もり

- レンダリングクォンタム: 128 samples = 2.9ms (Web Audio API 固定)
- AudioContext latencyHint: "interactive"
- 合計: 15〜35ms (ハードウェア・ブラウザ依存)

操作から発音までの遅延がこの範囲に収まるため、楽器的なリアルタイム操作が成立。
