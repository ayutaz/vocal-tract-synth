## 1. 日本語音素の分類と声道パラメータ

### 1.1 閉鎖子音（破裂音）

| 音素 | IPA | 調音位置 | 閉鎖区間(16点idx) | 閉鎖面積 | VOT(ms) |
|------|-----|----------|-------------------|----------|---------|
| か行 | /k/, /g/ | 軟口蓋 | idx 5-6 | 0.05 cm² | +25-50 / -60-0 |
| た行 | /t/, /d/ | 歯茎 | idx 2-3 | 0.05 cm² | +15-30 / -60-0 |
| ぱ行 | /p/, /b/ | 両唇 | idx 0-1 | 0.05 cm² | +15-25 / -60-0 |

時間フェーズ: 閉鎖(50-100ms) → バースト(5-15ms) → VOT → 母音遷移(30-50ms)

### 1.2 摩擦子音

| 音素 | IPA | 狭窄位置(16点idx) | 狭窄面積 | ノイズスペクトルピーク |
|------|-----|-------------------|----------|---------------------|
| さ行 | /s/ | idx 2-3 | 0.10-0.15 cm² | 4000-8000 Hz |
| し | /ɕ/ | idx 3-4 | 0.10-0.20 cm² | 2500-5000 Hz |
| は行 | /h/ | 声門 | 声門ノイズ | 声道共鳴依存 |
| ひ | /ç/ | idx 4-5 | 0.15-0.30 cm² | 2000-5000 Hz |
| ふ | /ɸ/ | idx 0-1 | 0.15-0.30 cm² | 1000-4000 Hz |
| ざ行 | /z/, /ʑ/ | idx 2-4 | 0.10-0.25 cm² | 有声+ノイズ混合 |

### 1.3 破擦音

| 音素 | IPA | 閉鎖位置 | 摩擦位置 | 閉鎖(ms) | 摩擦(ms) |
|------|-----|----------|----------|----------|----------|
| ち | /tɕ/ | idx 3-5 | idx 2-4 | 40 | 70 |
| つ | /ts/ | idx 1-3 | idx 1-2 | 35 | 60 |
| じ/ぢ | /dʑ/ | idx 3-5 | idx 2-4 | 30 | 50 |
| ず/づ | /dz/ | idx 1-3 | idx 1-2 | 25 | 40 |

### 1.4 鼻音

| 音素 | IPA | 口腔閉鎖位置 | 鼻腔ポート面積 |
|------|-----|-------------|---------------|
| ま行 | /m/ | idx 0-1 (唇) | 1.5-2.0 cm² |
| な行 | /n/ | idx 2-3 (歯茎) | 1.5-2.0 cm² |
| に | /ɲ/ | idx 3-4 (硬口蓋) | 1.5-2.0 cm² |
| ん+か行 | /ŋ/ | idx 5-6 (軟口蓋) | 1.5-2.0 cm² |

鼻腔管: 長さ約12cm、30区間（固定断面積、ハードコード）
3ポート接合: `k = (A_p*f_p + A_o*b_o + A_n*b_n) / A_sum` (乗算3回)

### 1.5 弾き音・半母音

| 音素 | IPA | 特徴 | 遷移時間 |
|------|-----|------|----------|
| ら行 | /ɾ/ | idx 2-3 瞬間接触(5-15ms) | 20-30ms全体 |
| や行 | /j/ | /i/形状からの高速遷移 | 50-80ms |
| わ行 | /w/ | /u/形状からの高速遷移 | 60-100ms |

### 1.6 特殊拍

| 記号 | 実装方法 |
|------|---------|
| っ (促音) | 後続子音の閉鎖/摩擦を1モーラ分延長 |
| ー (長音) | 前母音を1モーラ分延長 |
| ん (撥音) | 後続音素で異音: +ま行→[m], +な行→[n], +か行→[ŋ], 語末→[ɴ] |
| 拗音 | 子音の口蓋化(idx 3-4狭窄)→母音遷移 |

## 2. 必要なモデル拡張

### 2.1 MIN_AREA の引き下げ

現在 0.3 cm² → **0.01 cm²** に変更。完全閉鎖の模擬に必須。
- UI ドラッグの下限は 0.3 のまま維持（プログラム制御のみ 0.01 許可）
- r = (4.0-0.01)/(4.0+0.01) = 0.995 で数値的に安定

### 2.2 声道内ノイズ注入

摩擦音・バーストノイズ用。VocalTract の散乱ループ内で特定区間の前進波にノイズ加算。
```
f[k] += friction_gain * bandpass_noise
```
追加コスト: LCG(2ops) + Biquad(5ops) + 加算(1op) = **~8 ops/sample**

### 2.3 鼻腔分岐管

30区間の鼻腔管 + 口蓋帆（velum）ポートの3ポート接合。
- 追加計算量: +25%（29散乱 + 3方向接合）
- velum 閉鎖時はスキップ可能（母音発声時コストゼロ）

### 2.4 Worklet側サンプル精度補間

子音の5-20ms遷移はrAF(16ms)では不足。断面積キーフレームをWorkletに一括送信し、サンプル単位で線形補間。
```typescript
{ type: 'scheduleTransition'; targetAreas: number[]; durationSamples: number }
```

## 3. テキスト→音素変換

### 3.1 マッピング方式

テーブル駆動、最長一致（2文字→1文字 fallback）。約85エントリ。

```
入力: "こんにちは"
Step 1: 最長一致 → ['k','o','N','n','i','tɕ','i','h','a']
Step 2: 撥音後処理 N+n → [n]
Step 3: 持続時間割当
Step 4: 声道形状シーケンス生成
```

### 3.2 音素持続時間

| カテゴリ | 持続時間(ms) |
|---------|-------------|
| 母音 | 80-120 |
| 破裂子音 | 20-40 (+ 閉鎖60ms) |
| 摩擦子音 | 60-80 |
| 破擦子音 | 90 (閉鎖+摩擦) |
| 鼻子音 | 40-60 |
| 弾音 | 20-30 |
| 半母音 | 30-50 |
| 促音 | 80-120 (無音) |

## 4. 韻律モデル

### 4.1 F0制御（簡易ルールベース）

- デフォルト平板型: 1モーラ目=Low(110Hz), 2モーラ目以降=High(140Hz)
- 文末下降: 最終2-3モーラで 140→80Hz
- 疑問文: 文末 +50Hz 上昇
- Declination: `F0_base *= exp(-0.3 * t)`

### 4.2 強度パターン

| 要素 | 相対振幅 |
|------|---------|
| 母音 /a/ | 1.0 |
| 母音 /i,u/ | 0.7-0.85 |
| 鼻音 | 0.5-0.6 |
| 摩擦音 | 0.2-0.4 |
| 破裂バースト | 0.3-0.5 |
