# 声道シミュレーション音声合成アプリ - 要求定義書

## 1. プロジェクト概要

### 1.1 目的
声道の物理モデル（source-filter model）をリアルタイムに操作し、人工音声を生成するインタラクティブなWebアプリケーション。
声門（glottis）で生成された音源が、声道（vocal tract）というフィルタを通過することで母音等の音声になる仕組みを、直感的に触って体験・演奏できる「声の楽器」。

### 1.2 背景
- 調音音声合成（articulatory synthesis）/ source-filter synthesis の考え方に基づく
- Praat の vocal tract area function の発想を、演奏的・触覚的なUIに落とし込む
- 教育用デモ、実験用インターフェース、舞台作品の楽器としての活用を想定

### 1.3 技術方針
- ブラウザ上で完結するWebアプリ（サーバー不要）
- Web Audio API によるリアルタイム音声合成
- Canvas（または SVG）による可視化
- source-filter model: 声門音源 + 声道フィルタ（連結管モデル）

## 2. 機能要求

### 2.1 声道断面積エディタ（メインUI）

| 項目 | 詳細 |
|------|------|
| 表示方向 | 左 = 唇（lips）、右 = 声門（glottis） |
| 内部区間数 | 44区間の連結管（fs=44100Hzでの物理的に正しい離散化） |
| UI制御点数 | 16個（44区間へスプライン補間） |
| 操作方法 | 各区間の断面積をドラッグで変更可能 |
| 表示形式 | 縦軸 = 断面積、横軸 = 声道の位置（唇〜声門） |
| 制御点 | 各区間の境界に制御点（ドラッグ可能なポイント）を表示 |
| リアルタイム反映 | ドラッグ中にリアルタイムで音が変化する |

### 2.2 音源（Source）

| 項目 | 詳細 |
|------|------|
| 声門パルス | 周期的な声門波形（glottal pulse train）を生成 |
| ピッチ制御 | 基本周波数（F0）を調整可能 |
| ノイズ源 | ホワイトノイズ等による無声音源（ボタンで切替） |
| 音源切替 | 有声音（パルス）/ 無声音（ノイズ）の切替 |

### 2.3 フィルタ（声道モデル）

| 項目 | 詳細 |
|------|------|
| モデル | 連結管モデル（concatenated tube model） |
| アルゴリズム | Kelly-Lochbaum アルゴリズム（Smith式1乗算接合） |
| パラメータ | 各区間の断面積（area function）、44区間 |
| 壁面損失 | 各区間に減衰係数 mu≈0.999 を適用（音質に必須） |
| リアルタイム | 断面積変更時に即座にフィルタ特性が更新される |

### 2.4 母音プリセット

| 項目 | 詳細 |
|------|------|
| プリセット | あ (a)、い (i)、う (u)、え (e)、お (o) の5母音 |
| 動作 | ボタン押下で、対応する母音の声道断面積パターンに設定 |
| 遷移 | プリセット間の切替時、断面積が滑らかに補間遷移する（任意） |

### 2.5 特殊モード

| 項目 | 詳細 |
|------|------|
| フラット | 声道を均一な断面積（ストレート管）にリセット |
| ノイズ | 音源をノイズに切替 |

### 2.6 再生制御

| 項目 | 詳細 |
|------|------|
| 開始/停止 | 音声合成の開始・停止ボタン |
| 音量 | 出力音量の調整（任意） |

### 2.7 自動モード（Auto Sing）

| 項目 | 詳細 |
|------|------|
| 概要 | ランダムな母音列とピッチで自動的に「歌う」モード |
| 母音遷移 | ランダムに母音プリセットを選択し、声道形状を自動遷移 |
| ピッチ変化 | ランダム（または音階ベース）でF0を変化させる |
| 速度調整 | 遷移速度をスライダーで調整可能 |
| ON/OFF | 自動モードのトグルスイッチ |

### 2.8 スペクトル表示

| 項目 | 詳細 |
|------|------|
| FFTスペクトル | 合成音声のリアルタイム周波数スペクトルを表示 |
| フォルマント表示 | F1, F2, F3 の推定値を数値で表示 |
| 更新頻度 | リアルタイム（フレーム毎） |

## 5. 音声合成アルゴリズム概要

### 5.1 Source（音源生成）
1. 基本周波数 F0 に基づき、声門パルス波形を周期的に生成
2. 声門波形モデル: Rosenberg 型パルス、LF モデル、または簡易三角波など
3. ノイズモードではホワイトノイズを音源とする

### 5.2 Filter（声道フィルタリング）
1. 声道を N 個（≈16）の等長円筒管に分割
2. 各管の断面積 A[i] から、隣接管境界の反射係数を計算:
   ```
   r[i] = (A[i+1] - A[i]) / (A[i+1] + A[i])
   ```
3. Kelly-Lochbaum アルゴリズムにより、進行波・反射波を各サンプルごとに伝搬
4. 唇端から放射される進行波が出力音声となる
5. 唇端の放射特性（radiation impedance）を近似的に適用

### 5.3 フォルマント推定
1. 出力音声に FFT を適用しスペクトルを計算
2. スペクトルのピーク検出により F1, F2, F3 を推定
3. または声道伝達関数から直接フォルマントを計算

## 6. 母音プリセット参考値（断面積パターン）

各母音に対応する声道断面積の典型値（cm^2、唇側→声門側）:

| 母音 | 特徴 |
|------|------|
| あ (a) | 咽頭部狭め、口腔部広め（開口大） |
| い (i) | 口腔前部狭め、咽頭部広め |
| う (u) | 唇部狭め（円唇）、全体的にやや狭い |
| え (e) | 口腔前部やや狭め、咽頭部広め（「い」より開き気味） |
| お (o) | 唇部狭め（円唇）、咽頭部広め |

具体的な数値は Fant (1960) や Praat の文献値を参考に調整。

## 7. 将来的な拡展可能性（スコープ外だが設計で考慮）

方向性はTTS寄り（テキスト/音素 → 声道操作 → 音声生成）。マイク入力・逆推定は対象外。

1. **テキスト入力からの発話**
   - 日本語テキスト → 音素列 → 声道形状シーケンス → 連続音声
2. **歌唱技法の制御**
   - チェストボイス / ファルセット / ブレシー等の声門モデル切替
3. **MIDI / OSC 対応**
   - 外部コントローラーからのピッチ・母音制御

## 8. 成功基準

- [ ] 声道断面積をドラッグで変更でき、リアルタイムに音が変化する
- [ ] 5母音（あいうえお）のプリセットが正しい母音に聞こえる
- [ ] スペクトル表示で母音ごとのフォルマント構造の違いが視認できる
- [ ] 自動モードで母音とピッチがランダムに遷移し「歌っている」ように聞こえる
- [ ] ブラウザ上で低レイテンシに動作する（操作に対する音の遅延が気にならない）
